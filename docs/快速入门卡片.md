## 🚀 启动系统

```bash

cd 你的项目路径

# 2. 激活虚拟环境
venv\Scripts\activate

streamlit run app/Home.py

## 📈 买入检查清单

执行买入前，逐项确认：

- [ ] 大盘是绿灯？
- [ ] 信号强度 ≥ 83？
- [ ] 当前持仓 < 5只？
- [ ] 买入金额 ≤ 总资金11%？
- [ ] 股数是100的整数倍？

**全部打勾才能买入！**

---

## 📉 卖出检查清单

触发任一条件即卖出：

| 条件 | 触发规则 | 必须执行？ |
|------|----------|------------|
| 止损 | 亏损 ≥ 4.6% | ✅ 必须！ |
| 止盈 | 盈利 ≥ 22% | ✅ 建议 |
| 移动止盈 | 盈利曾达9%，回撤2.8% | ✅ 建议 |
| RSI超买 | RSI > 80 且盈利 | ⚠️ 建议 |
| 趋势反转 | MA5 < MA20 且亏损 | ⚠️ 建议 |
| 持仓超时 | 持仓 ≥ 15天 | ⚠️ 考虑 |

---

## 🔢 关键参数

```
止损: -4.6%    止盈: +22%

最大持仓: ≤5只
