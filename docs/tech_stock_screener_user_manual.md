## 系统简介

### 什么是科技股筛选系统？

科技股筛选系统是一个自动化的股票筛选工具，帮助您从A股市场中筛选出高质量的科技股票。系统采用多维度评估方法，综合考虑行业属性、财务健康、市场表现和竞争优势等因素。

### 系统特点

| 特点 | 说明 |
|------|------|
| 🎯 精准筛选 | 基于8大科技行业的关键词匹配 |
| 📊 多维评估 | 财务、市场、行业、定性四维度评分 |
| ⚡ 高效处理 | 支持全市场5000+股票快速筛选 |
| 🛡️ 风险控制 | 内置风险评估和预警机制 |
| 🔄 自动更新 | 支持定期自动更新股票池 |

### 适用场景

- 科技股投资研究
- 股票池构建和优化
- 行业分析和对比
- 投资组合管理

## 功能详解

### 1. 行业筛选

系统支持8大科技细分行业：

| 行业 | 代表性公司 | 关键词示例 |
|------|------------|------------|
| 半导体 | 中芯国际、韦尔股份 | 芯片、集成电路、晶圆 |
| 人工智能 | 科大讯飞、海康威视 | AI、机器学习、算法 |
| 5G通信 | 中兴通讯、烽火通信 | 5G、基站、光通信 |
| 新能源科技 | 宁德时代、隆基绿能 | 锂电池、光伏、储能 |
| 消费电子 | 立讯精密、歌尔股份 | 智能手机、可穿戴 |
| 软件服务 | 用友网络、金山办公 | 软件、云计算、SaaS |
| 生物医药科技 | 迈瑞医疗、药明康德 | 医疗器械、基因 |
| 智能制造 | 汇川技术、埃斯顿 | 工业自动化、机器人 |

### 2. 财务筛选

系统评估以下财务指标：

**盈利能力指标**
- ROE（净资产收益率）：衡量股东投资回报
- ROA（总资产收益率）：衡量资产利用效率
- 毛利率：衡量产品盈利能力
- 净利率：衡量最终盈利水平

**成长性指标**
- 营收增长率：衡量业务扩张速度
- 净利润增长率：衡量盈利增长能力
- 研发投入占比：衡量创新投入力度

**财务稳健性指标**
- 资产负债率：衡量财务杠杆
- 流动比率：衡量短期偿债能力
- 现金流量比率：衡量现金流健康度

**估值指标**
- PE（市盈率）：衡量估值水平
- PB（市净率）：衡量资产估值
- PEG：衡量估值与成长的匹配度

### 3. 市场筛选

系统评估以下市场指标：

**流动性指标**
- 总市值：公司规模
- 流通市值：可交易规模
- 日均成交额：交易活跃度
- 换手率：交易频率

**稳定性指标**
- 年化波动率：价格波动程度
- 最大回撤：历史最大跌幅
- 涨跌停频率：极端波动频率

### 4. 综合评分

系统采用加权评分模型：

```
综合得分 = 财务健康度 × 35% + 成长潜力 × 25% + 市场表现 × 20% + 竞争优势 × 20%
```

**评级标准**

| 评级 | 分数 | 投资建议 |
|------|------|----------|
| AAA | 90-100 | ⭐⭐⭐⭐⭐ 顶级，强烈推荐 |
| AA | 80-89 | ⭐⭐⭐⭐ 优秀，重点关注 |
| A | 70-79 | ⭐⭐⭐ 良好，可纳入观察 |
| BBB | 60-69 | ⭐⭐ 中等偏上 |
| BB | 50-59 | ⭐ 中等 |
| B | 40-49 | 中等偏下 |
| C | <40 | 较差，不推荐 |

## 操作指南

### 基本操作

#### 1. 运行完整筛选

```bash
# 使用默认参数
python tools/tech_stock_pool_expander.py

# 指定目标数量
python tools/tech_stock_pool_expander.py --target 100

# 指定最低评分
python tools/tech_stock_pool_expander.py --min-score 65
```

#### 2. 查看当前股票池

```bash
python tools/screen_tech_stocks.py --show-pool
```

#### 3. 更新股票池

```bash
python tools/tech_stock_pool_expander.py --update
```

### 高级操作

#### 1. 自定义筛选标准

编辑 `config/settings.py` 文件：

```python
# 财务筛选标准
FINANCIAL_CRITERIA = {
    'min_roe': 10.0,          # 提高ROE要求
    'min_gross_margin': 25.0, # 提高毛利率要求
    'max_debt_ratio': 50.0,   # 降低负债率限制
}
```

#### 2. 调整评分权重

```python
# 评分权重
SCORING_WEIGHTS = {
    'financial_health': 0.40,      # 增加财务权重
    'growth_potential': 0.30,      # 增加成长权重
    'market_performance': 0.15,    # 降低市场权重
    'competitive_advantage': 0.15, # 降低定性权重
}
```

#### 3. 添加行业关键词

```python
from core.stock_screener import get_industry_screener, TechIndustry

screener = get_industry_screener()
screener.add_keyword(TechIndustry.AI, "大模型")
screener.add_keyword(TechIndustry.SEMICONDUCTOR, "先进封装")
```

## 常见问题

### Q1: 筛选结果为什么和预期不同？

**A:** 可能原因：
1. 数据更新延迟：财务数据通常按季度更新
2. 筛选标准差异：检查配置文件中的筛选参数
3. 行业分类差异：系统基于关键词匹配，可能与其他分类不同

**解决方案：**
- 检查数据更新时间
- 调整筛选参数
- 手动添加行业关键词

### Q2: 为什么某只股票没有被选中？

**A:** 可以通过以下方式排查：

```python
from core.stock_screener import get_comprehensive_scorer

scorer = get_comprehensive_scorer()
# 获取单只股票的详细评分
score = scorer.score_stock(stock_row)
print(f"财务得分: {score.financial_score}")
print(f"市场得分: {score.market_score}")
print(f"失败原因: {score.weaknesses}")
```

### Q3: 如何提高筛选速度？

**A:** 
1. 启用缓存：减少重复数据获取
2. 使用并行处理：加速批量计算
3. 减少筛选范围：先进行粗筛

### Q4: 评分低的股票一定不好吗？

**A:** 不一定。评分是基于当前数据的综合评估：
- 新上市公司可能数据不完整
- 周期性行业可能暂时表现不佳
- 某些指标可能不适用于特定行业

建议结合自己的研究判断。

### Q5: 如何处理筛选结果？

**A:** 建议流程：
1. 重点关注AA级以上股票
2. 查看详细评分和风险提示
3. 结合基本面研究
4. 分散投资，控制风险
5. 定期复查和调整

## 最佳实践

### 1. 定期更新

建议每周更新一次股票池，保持数据新鲜度。

### 2. 分散投资

- 单一行业占比不超过25%
- 单只股票占比不超过5%
- 覆盖至少5个科技细分行业

### 3. 关注风险提示

系统会自动识别以下风险：
- 集中度风险
- 流动性风险
- 波动性风险
- 财务风险

### 4. 结合基本面研究

系统筛选是第一步，建议：
- 阅读公司年报和研报
- 关注行业发展趋势
- 了解公司竞争优势
- 评估管理层能力

### 5. 动态调整

市场环境变化时，适当调整：
- 牛市：可适当放宽估值标准
- 熊市：提高财务稳健性要求
- 行业轮动：关注新兴行业机会
