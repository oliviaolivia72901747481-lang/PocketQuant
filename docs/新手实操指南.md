## 系统概述

### 这是什么系统？

MiniQuant-Lite 是一套面向 A 股个人投资者的**量化投资辅助系统**，专注于科技股板块。

**核心功能：**
- 🔍 **智能选股**：从60只精选科技股中筛选买入机会
- 📊 **信号生成**：基于技术指标自动生成买卖信号
- 🛡️ **风险控制**：大盘滤网、止损止盈、持仓管理
- 📈 **回测验证**：历史数据验证策略有效性

### 适合谁使用？

- 有一定股票基础知识的个人投资者
- 希望用系统化方法辅助投资决策
- 能够严格执行交易纪律
- 资金量在 5-20 万之间

### 不适合谁？

- 完全没有股票知识的新手（建议先学习基础）
- 追求暴富、不愿止损的投资者
- 无法每天花 10-15 分钟操作的人

## 环境准备

### 第一步：安装 Python

1. 下载 Python 3.10 或更高版本：https://www.python.org/downloads/
2. 安装时勾选 "Add Python to PATH"
3. 验证安装：打开命令行，输入 `python --version`

### 第二步：安装依赖

打开命令行，进入项目目录，执行：

```bash
# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 第三步：验证安装

```bash
python -c "import streamlit; import akshare; print('安装成功！')"
```

## 首次启动

### 启动系统

在项目目录下执行：

```bash
streamlit run app/Home.py
```

系统会自动打开浏览器，显示首页。

### 首页说明

首页会显示以下信息：

1. **资金红绿灯** 🚦
   - 🟢 绿灯：资金充足，可以正常交易
   - 🟡 黄灯：资金偏少，注意手续费磨损
   - 🔴 红灯：资金不足，建议暂停交易

2. **数据新鲜度警告** ⚠️
   - 如果数据过期，会显示红色警告
   - 需要先更新数据才能使用

3. **科技股数据警告** 🔬
   - 如果科技股数据不完整，会显示警告
   - 需要下载缺失数据

4. **大盘状态** 📊
   - 显示沪深300与MA20的关系
   - 大盘滤网生效时建议空仓

## 数据准备

### 为什么需要下载数据？

系统需要股票历史数据来计算技术指标和生成信号。首次使用必须下载数据。

### 下载科技股数据

**方法一：通过数据管理页面（推荐）**

1. 点击左侧菜单 "📊 数据管理"
2. 找到 "🔬 科技股数据专区"
3. 查看数据完整性统计
4. 点击 "🚀 开始下载科技股数据"
5. 等待下载完成（约 5-10 分钟）

**方法二：通过科技股页面**

1. 点击左侧菜单 "🔬 Tech Stock"
2. 如果数据不完整，会显示警告
3. 点击下载按钮

### 数据更新频率

- **每日更新**：交易日收盘后（15:30 后）更新一次
- **周末不需要更新**：周末数据不变
- **建议时间**：每天 19:00-21:00 更新并查看信号

## 每日操作流程

### 标准作业程序 (SOP)

```
┌─────────────────────────────────────────────────────────┐
│                    每日操作流程                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  19:00-21:00  启动系统                                  │
│       ↓                                                 │
│  检查首页警告（数据、大盘状态）                          │
│       ↓                                                 │
│  更新数据（如需要）                                      │
│       ↓                                                 │
│  查看科技股页面                                          │
│       ↓                                                 │
│  检查大盘红绿灯                                          │
│       ├── 红灯 → 空仓观望，不操作                        │
│       └── 绿灯 → 继续下一步                              │
│       ↓                                                 │
│  查看买入信号                                            │
│       ├── 有信号 → 记录，准备次日买入                    │
│       └── 无信号 → 检查持仓卖出信号                      │
│       ↓                                                 │
│  查看卖出信号                                            │
│       ├── 有信号 → 记录，准备次日卖出                    │
│       └── 无信号 → 继续持有                              │
│       ↓                                                 │
│  设置券商条件单（可选）                                  │
│       ↓                                                 │
│  次日开盘执行                                            │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 为什么选择 19:00-21:00？

1. **数据完整**：收盘数据在 16:30 后稳定
2. **公告发布**：盘后公告基本发布完毕
3. **充足时间**：有时间分析和设置条件单
4. **避免冲动**：不在盘中操作，减少情绪影响

## 买入信号解读

### 进入科技股页面

点击左侧菜单 "🔬 Tech Stock"

### 理解大盘红绿灯

页面顶部会显示大盘状态：

| 状态 | 含义 | 操作建议 |
|------|------|----------|
| 🟢 绿灯 | 沪深300 > MA20 | 可以买入 |
| 🔴 红灯 | 沪深300 < MA20 | 禁止买入，空仓观望 |

**重要**：红灯时绝对不要买入！这是最重要的风控规则。

### 查看买入信号

如果大盘是绿灯，向下滚动查看买入信号列表：

| 字段 | 含义 |
|------|------|
| 股票代码 | 6位股票代码 |
| 股票名称 | 股票简称 |
| 信号强度 | 0-100分，越高越好 |
| 当前价格 | 最新收盘价 |
| RSI | 相对强弱指标 |
| 建议仓位 | 建议买入金额占比 |

### 信号强度解读

| 强度 | 颜色 | 含义 | 建议 |
|------|------|------|------|
| ≥90 | 🟢 绿色 | 强信号 | 优先买入 |
| 80-89 | 🟡 黄色 | 中等信号 | 可以买入 |
| <80 | ⚪ 灰色 | 弱信号 | 谨慎考虑 |

### 买入决策流程

```
收到买入信号
    ↓
检查信号强度 ≥ 83？
    ├── 否 → 放弃
    └── 是 → 继续
    ↓
检查当前持仓 < 5只？
    ├── 否 → 放弃（已满仓）
    └── 是 → 继续
    ↓
计算买入金额（≤总资金的11%）
    ↓
计算买入股数（100股整数倍）
    ↓
次日开盘买入
```

### 买入示例

假设：
- 总资金：100,000 元
- 信号股票：002371 北方华创
- 当前价格：150 元
- 信号强度：92

计算：
1. 最大买入金额 = 100,000 × 11% = 11,000 元
2. 可买股数 = 11,000 ÷ 150 = 73 股
3. 取整为 100 股整数倍 = 0 股（不够100股）
4. 或者增加资金买 100 股 = 15,000 元

决策：如果资金允许，买入 100 股，成本 15,000 元。

## 卖出信号解读

### 卖出信号类型

系统会自动检测以下卖出条件：

| 类型 | 触发条件 | 优先级 |
|------|----------|--------|
| 🔴 止损 | 亏损 ≥ 4.6% | 最高 |
| 🟢 止盈 | 盈利 ≥ 22% | 高 |
| 🟡 移动止盈 | 盈利曾达9%，回撤2.8% | 高 |
| 🟠 RSI超买 | RSI > 80 且盈利 | 中 |
| 🔵 趋势反转 | MA5 < MA20 且亏损 | 中 |
| ⚪ 持仓超时 | 持仓 ≥ 15天 | 低 |

### 卖出信号颜色

| 颜色 | 含义 | 操作 |
|------|------|------|
| 🔴 红色 | 紧急卖出 | 必须立即执行 |
| 🟠 橙色 | 建议卖出 | 强烈建议执行 |
| 🟡 黄色 | 可以卖出 | 建议执行 |
| 🟢 绿色 | 止盈卖出 | 锁定利润 |

### 卖出决策流程

```
收到卖出信号
    ↓
确认信号类型
    ↓
止损信号？
    ├── 是 → 必须执行！不要犹豫！
    └── 否 → 继续判断
    ↓
止盈/移动止盈信号？
    ├── 是 → 建议执行，锁定利润
    └── 否 → 继续判断
    ↓
其他信号？
    └── 根据情况决定
    ↓
次日开盘卖出
```

### 止损的重要性

**止损是最重要的风控规则！**

- 亏损 4.6% 必须止损
- 不要抱有侥幸心理
- 不要补仓摊低成本
- 小亏是为了避免大亏

## 持仓管理

### 查看当前持仓

1. 点击左侧菜单 "📊 Position"
2. 查看当前持仓列表

### 持仓信息说明

| 字段 | 含义 |
|------|------|
| 股票代码 | 持仓股票代码 |
| 买入价格 | 买入成本价 |
| 当前价格 | 最新价格 |
| 盈亏比例 | 当前盈亏百分比 |
| 持仓天数 | 已持有天数 |
| 止损价 | 触发止损的价格 |
| 止盈价 | 触发止盈的价格 |

### 仓位控制规则

| 规则 | 数值 | 说明 |
|------|------|------|
| 单只仓位 | ≤11% | 单只股票最大占比 |
| 最大持仓 | ≤5只 | 同时持有股票数量 |
| 现金缓冲 | 5% | 保留部分现金 |

### 添加持仓

买入股票后，需要在系统中记录：

1. 进入 Position 页面
2. 点击 "添加持仓"
3. 填写股票代码、买入价格、股数、买入日期
4. 保存

### 删除持仓

卖出股票后，需要删除记录：

1. 进入 Position 页面
2. 找到已卖出的股票
3. 点击 "删除"

## 常见问题

### Q1: 系统启动失败怎么办？

**检查步骤：**
1. 确认 Python 版本 ≥ 3.10
2. 确认已激活虚拟环境
3. 确认已安装所有依赖
4. 查看错误信息，搜索解决方案

### Q2: 数据下载失败怎么办？

**可能原因：**
1. 网络连接问题 → 检查网络
2. AkShare 接口限流 → 等待几分钟重试
3. 数据源维护 → 稍后再试

### Q3: 没有买入信号怎么办？

**正常情况：**
- 不是每天都有信号
- 没有信号说明没有好的买入机会
- 耐心等待，不要强行买入

### Q4: 大盘红灯但有买入信号怎么办？

**答案：不买！**
- 大盘红灯时禁止买入
- 这是最重要的风控规则
- 等大盘转绿再考虑

### Q5: 止损后股票又涨回来了怎么办？

**正确态度：**
- 止损是正确的决定
- 不要后悔，不要追回
- 长期来看，止损保护了本金
- 下次机会还会有

### Q6: 信号强度不够高要不要买？

**建议：**
- 信号强度 < 83 不建议买入
- 宁可错过，不要做错
- 等待更好的机会

### Q7: 可以同时持有超过5只股票吗？

**答案：不建议**
- 5只是经过回测验证的最优数量
- 超过5只会分散精力和资金
- 严格遵守规则

## 风险提示

### ⚠️ 重要声明

1. **本系统仅供学习研究使用，不构成投资建议**
2. **股市有风险，投资需谨慎**
3. **历史表现不代表未来收益**
4. **请用闲置资金投资，不要借钱炒股**

| 禁止事项 | 原因 |
|----------|------|
| 不止损 | 小亏变大亏 |
| 超仓位 | 风险过于集中 |
| 追涨杀跌 | 情绪化交易 |
| 频繁交易 | 手续费磨损 |
| 借钱炒股 | 影响生活 |

### ✅ 必须遵守

| 规则 | 说明 |
|------|------|
| 严格止损 | 亏损4.6%必须止损 |
| 控制仓位 | 单只≤11%，总共≤5只 |
| 按信号操作 | 不要主观判断 |
| 大盘红灯空仓 | 最重要的风控 |
| 每日检查 | 保持纪律 |
