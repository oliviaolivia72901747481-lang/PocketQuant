# 科技股实盘监控模板使用说明

## 📁 文件说明

模板文件：`科技股实盘监控模板_v11.2_YYYYMMDD.xlsx`

包含以下工作表：
1. **交易记录** - 记录每笔买卖交易
2. **当前持仓** - 实时持仓监控
3. **绩效统计** - 月度/年度绩效汇总
4. **策略参数** - v11.2策略参数配置
5. **使用说明** - 模板使用指南
6. **股票池** - 60只科技股列表

---

## 📊 工作表详解

### 1. 交易记录表

记录每笔完整的交易（买入+卖出）。

| 字段 | 说明 | 填写方式 |
|------|------|----------|
| 序号 | 自动编号 | 自动 |
| 交易日期 | 卖出日期 | 手动填写 |
| 股票代码 | 6位代码 | 手动填写 |
| 股票名称 | 股票简称 | 手动填写 |
| 交易方向 | 买入/卖出 | 手动填写 |
| 买入价格 | 买入成交价 | 手动填写 |
| 卖出价格 | 卖出成交价 | 手动填写 |
| 股数 | 交易股数 | 手动填写 |
| 买入金额 | 买入价×股数 | 手动填写 |
| 卖出金额 | 卖出价×股数 | 手动填写 |
| 盈亏金额 | 卖出-买入 | **自动计算** |
| 盈亏比例 | 盈亏/买入 | **自动计算** |
| 持仓天数 | 持有天数 | 手动填写 |
| 卖出原因 | 见下表 | 手动填写 |
| 备注 | 其他说明 | 手动填写 |

**卖出原因选项：**
- 止损 - 亏损≥4.5%
- 止盈 - 盈利≥20%
- 移动止盈 - 盈利曾达9%，回撤2.8%
- RSI超买 - RSI>80
- 趋势反转 - MA5<MA20且亏损
- 持仓超时 - 持仓≥15天

## 📋 操作流程

### 每日开盘前（9:00-9:25）

1. 打开系统查看买入信号
2. 确认信号强度≥85
3. 检查当前持仓数量（最多5只）
4. 准备买入资金（单只≤12%仓位）

### 买入操作

1. 按信号买入股票
2. 在"当前持仓"表添加记录：
   - 填写股票代码、名称
   - 填写买入日期、价格、股数
   - 其他字段自动计算

### 每日收盘后（15:00后）

1. 更新所有持仓的"当前价格"
2. 更新"最高价"（如果创新高）
3. 检查卖出条件：
   - 当前价格 ≤ 止损价 → 次日止损
   - 当前价格 ≥ 止盈价 → 次日止盈
   - 最高盈利≥9% 且 当前盈利回撤2.8% → 移动止盈
   - RSI > 80 → RSI超买卖出
   - 持仓天数 ≥ 15 → 持仓超时卖出

### 卖出操作

1. 执行卖出
2. 从"当前持仓"表删除记录
3. 在"交易记录"表添加完整交易记录

### 每月末

1. 汇总本月交易数据
2. 填写"绩效统计"表的月度绩效
3. 复盘分析，总结经验

## ⚠️ 风险控制要点

### 止损纪律（最重要！）

```
当前价格 ≤ 买入价格 × 0.955 → 必须止损！
```

- 不要抱有侥幸心理
- 不要补仓摊低成本
- 严格执行，保护本金

### 仓位控制

- 单只股票：≤12%仓位
- 最大持仓：≤5只股票
- 总仓位：根据市场情况调整

### 移动止盈

```
如果 最高盈利 ≥ 9%：
    移动止盈价 = 最高价 × (1 - 2.8%)
    当前价格 ≤ 移动止盈价 → 卖出
```

## 💡 使用建议

1. **严格执行策略**：不要主观判断，按信号操作
2. **每日更新数据**：保持数据准确，及时发现风险
3. **定期复盘**：每月分析交易记录，总结经验
4. **控制情绪**：亏损时不要冲动，盈利时不要贪婪
5. **持续学习**：关注市场变化，优化策略
