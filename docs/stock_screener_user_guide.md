# 股票筛选引擎用户指南

## 简介

股票筛选引擎是一个自动化的科技股筛选工具，帮助您从全市场股票中筛选出高质量的科技股票。

## 功能特点

- **多维度筛选**: 行业、财务、市场表现等多角度筛选
- **智能评分**: 综合评分系统，自动评级
- **风险控制**: 内置风险评估和预警机制
- **自动更新**: 支持定期自动更新股票池
- **高性能**: 并行处理和缓存优化

## 快速入门

### 1. 运行筛选工具

```bash
python tools/tech_stock_pool_expander.py
```

### 2. 查看筛选结果

筛选完成后，结果将保存在 `data/` 目录下：
- `tech_stock_candidates.csv`: 候选股票列表
- `screening_report.md`: 筛选报告

### 3. 更新股票池

```bash
python tools/update_stock_pool.py
```

## 筛选流程

### 第一步：数据获取

系统从多个数据源获取全市场股票数据：
- 股票基本信息
- 财务指标数据
- 市场交易数据

### 第二步：数据清洗

自动清洗数据：
- 过滤非主板股票（保留沪深主板）
- 移除ST/*ST股票
- 处理缺失值和异常值

### 第三步：行业筛选

识别科技行业股票：
- 半导体
- 人工智能
- 云计算
- 新能源
- 5G通信
- 物联网
- 网络安全
- 大数据

### 第四步：财务筛选

基于财务指标筛选：
- ROE ≥ 8%
- 毛利率 ≥ 20%
- 营收增长率 > 0
- 负债率 < 70%

### 第五步：市场筛选

基于市场表现筛选：
- 市值 ≥ 50亿
- 日成交额 ≥ 5000万
- 换手率适中

### 第六步：综合评分

多维度综合评分：
- 财务健康度 (35%)
- 成长潜力 (25%)
- 市场表现 (20%)
- 竞争优势 (20%)

### 第七步：质量验证

验证筛选结果：
- 数据质量检查
- 风险评估
- 行业分散度检查

## 评分说明

### 评级标准

| 评级 | 分数范围 | 说明 |
|------|----------|------|
| AAA | 90-100 | 顶级，强烈推荐 |
| AA | 80-89 | 优秀，推荐关注 |
| A | 70-79 | 良好，可纳入观察 |
| BBB | 60-69 | 中等偏上 |
| BB | 50-59 | 中等 |
| B | 40-49 | 中等偏下 |
| C | <40 | 较差，不推荐 |

### 评分维度

1. **财务健康度** (35%)
   - 盈利能力：ROE、ROA、毛利率
   - 成长性：营收增长、利润增长
   - 稳健性：负债率、流动比率

2. **成长潜力** (25%)
   - 行业匹配度
   - 研发投入
   - 业务前景

3. **市场表现** (20%)
   - 流动性：成交额、换手率
   - 稳定性：波动率、回撤

4. **竞争优势** (20%)
   - 行业地位
   - 技术壁垒
   - 品牌优势

## 风险提示

### 风险类型

1. **集中度风险**: 单一行业占比过高
2. **流动性风险**: 成交不活跃
3. **波动性风险**: 价格波动大
4. **财务风险**: 负债率高或亏损

### 风险等级

- **低风险**: 可放心持有
- **中等风险**: 需要关注
- **高风险**: 谨慎操作
- **严重风险**: 建议回避

## 常见问题

### Q: 筛选结果多久更新一次？

A: 建议每周更新一次，或在市场发生重大变化时更新。

### Q: 如何调整筛选标准？

A: 修改 `config/settings.py` 中的筛选参数。

### Q: 评分低的股票一定不好吗？

A: 不一定。评分是基于当前数据的综合评估，低分可能是因为数据不完整或不符合当前筛选标准。

### Q: 如何处理筛选结果？

A: 建议：
1. 重点关注AA级以上股票
2. 结合自己的研究判断
3. 分散投资，控制风险
