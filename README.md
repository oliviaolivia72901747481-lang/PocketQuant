# 📈 MiniQuant-Lite

**轻量级 A 股量化投资辅助系统 —— 5.5 万本金的「运钞车」**

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red.svg)](https://streamlit.io/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

MiniQuant-Lite 是一套面向 A 股个人投资者的轻量级量化投资辅助系统。系统基于 Python 构建，使用免费开源数据接口（AkShare），提供选股筛选、策略回测、每日交易信号生成和持仓管理功能。

> ⚠️ **风险提示**：本系统仅供学习研究使用，不构成投资建议。股市有风险，投资需谨慎。历史表现不代表未来收益。

## ⚠️ 重要免责声明

### 幸存者偏差警告

**本系统的回测结果可能存在幸存者偏差（Survivorship Bias）：**

1. **静态股票池问题**：回测基于当前存续的股票，未包含历史上已退市、暂停上市或被 ST 的标的
2. **数据局限性**：AkShare 提供的历史数据仅包含当前可交易的股票
3. **结果偏高风险**：由于排除了表现最差的退市股票，回测收益率可能被高估

**建议**：
- 将回测结果视为参考上限，实际收益可能低于回测
- 关注最大回撤和胜率等风险指标，而非仅看收益率
- 小资金实盘时保持谨慎，严格执行止损纪律

## ✨ 核心特性

### 🛡️ 保命模块（风控优先）

- **大盘滤网** - 沪深300 < MA20 时强制空仓，规避系统性风险
- **财报窗口期检测** - 财报披露前后 3 天自动剔除，避免财报黑天鹅
- **Smart Sizer** - 5% 现金缓冲、5 元低消预警、最大持仓限制
- **涨跌停板检测** - 一字板禁止交易，避免回测结果虚高
- **硬止损机制** - 亏损达到阈值自动触发卖出信号

### 💰 盈利模块（双策略支持）

- **RSI 超卖反弹策略**（默认）- 适合震荡行情，快进快出
  - 买入：RSI < 30（超卖区反弹）
  - 卖出：RSI > 70 或 止损 -5% 或 止盈 +15%
  
- **RSRS 阻力支撑策略** - 基于市场情绪的趋势策略
  - 买入：RSRS 标准分 > 0.7（市场情绪好）
  - 卖出：RSRS 标准分 < -0.7 或 止损 -6%

### 📊 可视化界面

- **Streamlit Dashboard** - 无需编写代码即可使用
- **数据管理** - 一键下载、更新、清空缓存
- **策略回测** - 可视化回测结果，参数敏感性分析
- **每日信号** - 早安确认清单，新闻链接快查
- **持仓管理** - 持仓跟踪、盈亏计算、卖出信号提醒

### 🔬 参数敏感性分析

- **热力图可视化** - 直观展示参数组合的收益分布
- **鲁棒性诊断** - 检测策略是否过拟合
- **最优参数推荐** - 自动找出最佳参数组合

## 📖 使用指南

### 标准作业程序 (SOP)

**推荐运行时间：交易日晚上 19:00 - 21:00**

```
晚上 19:00-21:00
    ↓
运行系统生成信号
    ↓
点击新闻链接，人眼扫一遍标题（10秒）
    ↓
将信号放入券商 APP 的"条件单"
    ↓
安心睡觉
    ↓
次日 9:25 前完成早安确认清单
    ↓
开盘自动执行
```

### 1. 数据管理

1. 进入「📊 数据管理」页面
2. 配置股票池（默认使用自选股列表）
3. 设置日期范围（建议下载最近 1 年数据）
4. 点击「开始下载」

> 💡 **提示**：首次使用需要下载历史数据，耗时约 1-5 分钟（取决于股票数量）

### 2. 策略回测

1. 进入「🧪 策略回测」页面
2. 选择策略：
   - **RSI 超卖反弹策略**（默认）- 适合震荡行情
   - **RSRS 阻力支撑策略** - 适合趋势行情
3. 配置回测参数（初始资金、日期范围等）
4. 点击「开始回测」
5. 查看回测结果：
   - 核心风控指标（胜率、最大回撤、盈亏比）
   - 收益指标（总收益率、年化收益率）
   - 收益分布图
   - 详细战绩表
6. 展开「参数敏感性分析」检测策略鲁棒性

### 3. 每日信号

1. 进入「📡 每日信号」页面
2. 查看持仓卖出信号（如有持仓）
3. 查看大盘状态（沪深300 vs MA20）
4. 选择策略（默认 RSI 超卖反弹策略）
5. 点击「生成今日信号」
6. 对每个信号：
   - 点击新闻链接，快速扫一眼标题
   - 注意财报窗口期警告
   - 注意高费率预警
7. 将确认无误的信号放入券商条件单

### 4. 持仓管理

1. 进入「💼 持仓管理」页面
2. 查看卖出信号提醒：
   - 🚨 止损信号（亏损 ≥ 6%）- 建议立即卖出
   - ⚠️ 策略卖出信号（RSRS < -0.7 或 RSI > 70）
3. 查看持仓列表及盈亏状态
4. 添加新持仓：输入股票代码、买入价格、数量、策略
5. 导出持仓记录为 CSV

## 📊 策略说明

### RSI 超卖反弹策略（默认）

**核心理念**：适合震荡行情，快进快出，积累小胜为大胜

#### 买入条件
- RSI < 30（超卖区反弹）

#### 卖出条件（任一满足）
- RSI > 70（超买止盈）
- 亏损达到 -5%（硬止损）
- 盈利达到 +15%（止盈）

---

### RSRS 阻力支撑策略

**核心理念**：基于阻力位和支撑位的变化关系判断市场情绪

#### 计算方法
1. 取过去 N=18 天的 High/Low 数据
2. 做线性回归，得到斜率 Beta
3. 将 Beta 标准化（Z-Score），与过去 M=600 天的历史比较
4. 得到 RSRS 标准分

#### 买入条件
- RSRS 标准分 > 0.7（市场情绪处于历史最好的 25%）

#### 卖出条件（任一满足）
- RSRS 标准分 < -0.7（市场情绪转弱）
- 亏损达到 -6%（硬止损）

## ❓ 常见问题

### Q: AkShare 数据获取失败怎么办？

A: 
1. 检查网络连接
2. 确认 AkShare 版本：`pip install akshare==1.12.0`
3. 避免在交易时段频繁请求
4. 如果持续失败，可能是接口临时维护，稍后重试

### Q: 为什么信号页面显示"今日无操作建议"？

A: 可能的原因：
1. 大盘滤网生效（沪深300 < MA20）
2. 没有股票满足买入条件
3. 股票数据未下载或数据不足

### Q: 如何添加自己的股票到股票池？

A: 
1. 在「📊 数据管理」页面的股票池管理区域添加
2. 或编辑 `config/stock_pool.py` 文件

### Q: 持仓管理的数据保存在哪里？

A: 持仓数据保存在 `data/positions.csv` 文件中，支持导出备份。
